Penny Pinchers
This project was inspired by my interest in finding a way to keep track of finances, which in turn gave me an opportunity to practice my ajax skills. My main focus for this website was to make sure that the page never had to fully reload and disrupt the user's experience. I also worked on my PHP skills by pulling variables from the chart and manipulating these variables with formulas. I wanted the user to be able to get rid of information they no longer needed as well, so I added in the delete button. I used the google chart API in order to give the user a visual representation of how their money was being distributed.
